## Списки

Чтобы добавить ненумерованные списки необходимо пункты выделить звездочкой (*). Например
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки нужно просто пронумеровать пункты. Например:
1. элемент 1
2. Элемент 2
